        # digitalclock

A simple digital clock with javascript.

## Features

*   Digital clock display.

## Tech Stack

*   HTML
*   JavaScript

## Folder Structure

```
.
├── clock.jpeg
└── index.html
```

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/AlvinGeorge-AG/digitalclock.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd digitalclock
    ```

## Usage

Open the `index.html` file in your web browser to view the digital clock.
